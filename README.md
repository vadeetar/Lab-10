---

## 📚 Теоретические сведения

### 🔹 FastAPI (Python)

* Современный асинхронный веб-фреймворк
* Высокая производительность за счёт async/await
* Использует Pydantic для валидации данных
* Автоматически генерирует Swagger-документацию

### 🔹 Gin (Go)

* Лёгкий и быстрый веб-фреймворк
* Поддержка middleware
* Использует контексты для обработки запросов
* Может быть расширен OpenAPI через swaggo

### 🔹 Способы взаимодействия сервисов

* REST API (реализовано)
* WebSocket (реализовано)
* gRPC (рассмотрено теоретически как более производительный вариант)

## 🚀 Запуск проекта

### 🔹 1. Запуск Go-сервиса

```bash
cd go-service
go mod tidy
go run main.go
```

### 🔹 2. Запуск Python-сервиса

```bash
cd python-service
pip install -r requirements.txt
uvicorn main:app --reload
```

### 🔹 Swagger (FastAPI)

```
http://127.0.0.1:8000/docs
```

## 📊 Сравнение FastAPI vs Gin

| Параметр      | Go (Gin)           | Python (FastAPI) |
| ------------- | ------------------ | ---------------- |
| Скорость      | Высокая            | Средняя          |
| RPS           | ~1200              | ~600             |
| Память        | Низкое потребление | Выше             |
